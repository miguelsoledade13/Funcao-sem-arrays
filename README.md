# Lista de Exercícios - Vetores em JavaScript

Este repositório contém a resolução de quatro exercícios de vetores desenvolvidos em JavaScript.

Os exercícios foram implementados utilizando estruturas de repetição, condicionais e vetores, sem o uso de métodos de arrays como `push()`, `pop()`, `filter()`, `map()`, `reverse()`, `includes()`, entre outros.

## Exercícios Desenvolvidos

### 1. Nomes em Ordem Inversa

O programa lê 7 nomes de pessoas e os armazena em um vetor. Ao final, exibe todos os nomes na ordem inversa daquela em que foram informados.

**Exemplo:**

Entrada:

```text
Ana
Carlos
Maria
João
Pedro
Lucas
Julia
```

Saída:

```text
Julia
Lucas
Pedro
João
Maria
Carlos
Ana
```

---

### 2. Média de Números Reais

O programa lê 8 números reais, armazena-os em um vetor e calcula a média dos valores informados. Em seguida, exibe apenas os números que estão acima da média calculada.

**Exemplo:**

Entrada:

```text
10
20
30
40
50
60
70
80
```

Saída:

```text
Média: 45
50
60
70
80
```

---

### 3. Menores de Idade

O programa lê o nome e a idade de 9 pessoas, armazenando essas informações em dois vetores relacionados. Ao final, exibe apenas os dados das pessoas menores de idade.

**Exemplo:**

Entrada:

```text
Ana - 15
Carlos - 20
Maria - 17
```

Saída:

```text
Ana - 15 anos
Maria - 17 anos
```

---

### 4. Interseção de Vetores

O programa lê dois vetores de 5 números inteiros cada e cria um terceiro vetor contendo apenas os elementos que aparecem nos dois vetores, sem repetição.

**Exemplo:**

Entrada:

```text
Vetor 1: [1, 2, 3, 4, 5]
Vetor 2: [3, 4, 5, 6, 7]
```

Saída:

```text
[3, 4, 5]
```

---
